## About AVV Lab

AVV Lab is designed for people who want a practical way to learn visual performance and lighting fundamentals without stepping outside the browser. The repository bundles standalone HTML learning modules covering VJing, DMX, lighting design, After Effects, and Illustrator, so related subjects can be explored in a single place.

It fits well in classrooms, self-paced study, and fast reference use during creative work. Because the lessons include interactive elements, the platform helps make abstract ideas easier to test, compare, and understand directly in the browser.
